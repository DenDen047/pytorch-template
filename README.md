# Pytorch Project Template

## Usage

```sh
$ cd docker && docker compose up -d --build
$ docker compose down && docker compose up -d --build && docker exec -it [container_name] bash
```

```bash
$ pyenv install 3.10.16
$ pyenv local 3.10.16
$ python -m venv .venv
$ source .venv/bin/activate

## pip install ...

$ deactivate
```
